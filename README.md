# Java Minesweeper Game

This is a classic Minesweeper game I made using Java. I started this project as a console application, but I later added a Graphical User Interface (GUI) to make it more fun to play.

## Features
- **Graphic Design:** Uses Java Swing buttons instead of just text.
- **Random Mines:** The game puts mines in different places every time you play.
- **Win/Loss Logic:** The game knows when you hit a mine or when you win by opening all safe areas[cite: 2].

## How to Play
1. Download or clone the project.
2. Open it in your IDE .
3. Run the `Main.java` file.
4. Enter the row and column size in the terminal, and the game window will pop up!
